📌 Overview

Golden Hub is a modern, responsive, and visually engaging co-working space website designed to present the services, workspaces, pricing plans, team, and contact information of a professional co-working environment.

The website focuses on delivering a premium digital experience through a combination of modern user interface design, responsive layouts, interactive components, smooth animations, and bilingual support for Arabic and English users.

The project was developed as a front-end web project using standard web technologies and is deployed online using GitHub Pages.

---

🎯 Project Objectives

The main objectives of the project are to:

- Create a professional digital presence for Golden Hub.
- Present available workspaces and services in a clear and attractive way.
- Provide users with an intuitive and responsive browsing experience.
- Support both Arabic and English interfaces.
- Provide interactive booking and contact forms.
- Apply modern front-end development and UI/UX design principles.
- Ensure compatibility across desktop, tablet, and mobile devices.

---

✨ Main Features

🌐 Bilingual Interface

- Arabic and English language support.
- RTL and LTR layout direction handling.
- Dynamic text translation using "data-i18n" attributes.
- Language preference persistence using "localStorage".

🎬 Interactive Hero Section

- Multi-slide hero presentation.
- Automatic slide transitions.
- Manual navigation controls.
- Slide indicators and counter.
- Smooth visual transitions and animations.
- Responsive layout for different screen sizes.

🏢 Workspace Showcase

The website presents different workspace options, including:

- Individual workstations.
- Meeting rooms.
- Private offices.
- Open co-working areas.

Each workspace is presented through visually engaging cards and interactive effects.

💳 Pricing Section

- Presentation of available pricing plans.
- Monthly and yearly pricing options.
- Interactive plan selection.
- Responsive pricing cards.

📅 Booking System

- Modal-based booking form.
- User input validation.
- Required booking information fields.
- Animated success feedback after successful submission.

📩 Contact System

- Contact information section.
- Contact form.
- Form validation.
- Success feedback animation.
- Location and map section.

👥 About and Team Sections

- Company mission and vision.
- Core values.
- Team member presentation.
- Timeline section highlighting important milestones.

📱 Responsive Design

The website is designed to adapt to:

- Desktop computers.
- Laptops.
- Tablets.
- Mobile phones.

The responsive layout is implemented using CSS media queries, Flexbox, CSS Grid, and Bootstrap's responsive grid system.

🚀 Getting Started

Prerequisites

No special software or build tools are required to run the project.

A modern web browser is sufficient.

---

Running the Project Locally

Option 1: Open Directly

Open the following file in a web browser:

index.html

Option 2: Use a Local Development Server

For a better development experience, the project can be run using a local server.

For example:

python -m http.server 8000

Then open:

http://localhost:8000
